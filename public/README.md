# Simple HTML/CSS/JavaScript Job Management System

## Files Created:
- `public/index.html` - Main HTML file
- `public/styles.css` - All styling
- `public/app.js` - JavaScript logic

## How to Run:

### Option 1: Open Directly in Browser
1. Navigate to `c:/Users/thamaraiselvi/New/public/`
2. Double-click `index.html` to open in your browser

### Option 2: With Backend (Recommended)
1. Make sure the backend server is running:
   ```
   npm run server
   ```
2. Open the file in browser or use a simple HTTP server

### Features:
✅ Beautiful animated login page
✅ Admin Portal with dashboard and job creation
✅ Designer Portal with job viewing and file management
✅ Connects to Node.js backend (http://localhost:3000)
✅ No React/Vite - Pure HTML/CSS/JS!

## Notes:
- The backend server must be running for full functionality
- CORS is configured to allow connections from file:// protocol
- All styles are in one CSS file (no external dependencies)
